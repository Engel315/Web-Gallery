# Веб-галерея изображений

## Описание
Проект представляет собой веб-приложение для создания и просмотра галереи изображений.
## Возможности
- Хранение и просмотр изображений
- Сортировка изображений по папкам
## Требования
- Python 3.8+
- Django 4.x (должен быть установлен)
- SQLite (встроенная база данных Django)
- Pillow (для обработки изображений; должен быть установлен)

## Установка
1. Клонируйте репозиторий:
```
git clone https://github.com/Engel315/Web-Gallery.git
cd photogallery
```
2. Создайте и активируйте виртуальное окружение:
```
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
.venv\Scripts\activate     # Windows
```
3. Миграции базы данных:
```
python manage.py migrate
```
## Запуск проекта
```
python manage.py runserver
```
После запуска перейдите по адресу: http://127.0.0.1:8000/.
Порт 8000 можно заменить на другой, например: 8080 или 9000.

## Использование
1. После загрузки проекта смените директорию в терминале:
```
cd photoweb
```
2. Создайте супер-пользователя:
```
python manage.py createsuperuser
```
3. Запустите веб приложение:
```
python manage.py runserver
```
И перейдите по адресу: http://127.0.0.1:8000/admin/ (Порт может отличаться).
Войдите в аккаунт супер-пользователя. Теперь вы можете настраивать веб-приложение.
Чтобы выключить веб-приложение, нажмите в терминале комбинацию клавиш Ctrl+C.
## Ограничения
- Поддерживаются только форматы изображений, совместимые с Pillow (JPEG, PNG, GIF и др.);
- Максимальный размер изображения ограничен настройками Django (по умолчанию 2.5 МБ);
- Нет системы регистрации пользователей.
## Технологии
- Django - веб-фреймворк Python
- SQLite - встроенная реляционная база данных
- Pillow - библиотека для обработки изображений
